# 🎉 Бот «Кухня народов» — для вечеринки

Telegram-бот, который рандомно раздаёт парам кухни мира и тайные задания.

---

## Как запустить (пошагово)

### 1. Создай бота в Telegram

1. Открой Telegram и найди **@BotFather** (с синей галочкой)
2. Напиши ему `/newbot`
3. Придумай имя для бота, например: `Кухня Народов`
4. Придумай username, например: `kuhnya_narodov_bot` (должен заканчиваться на `bot`)
5. BotFather пришлёт тебе **токен** — длинная строка вида `1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ`
6. **Скопируй этот токен** — он понадобится дальше

### 2. Вставь токен в бота

Открой файл `bot.py` и найди строку:
```
BOT_TOKEN = "ВСТАВЬ_СЮДА_ТОКЕН_ОТ_BOTFATHER"
```
Замени на свой токен:
```
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrSTUvwxYZ"
```

### 3. Установи зависимости

Открой терминал (Terminal на Mac) и выполни:
```bash
cd /Users/maximkirnichniy/party-bot
pip install -r requirements.txt
```

Если `pip` не работает, попробуй:
```bash
pip3 install -r requirements.txt
```

### 4. Запусти бота

```bash
python bot.py
```
или
```bash
python3 bot.py
```

Должно появиться: `🎉 Бот запущен! Жду участников...`

**Бот работает пока терминал открыт. Не закрывай его!**

---

## Как пользоваться

### Для тебя (админ):
1. Напиши боту `/start` первым — ты станешь админом
2. У тебя появится панель с кнопками:
   - **📊 Кто зарегался** — посмотреть сколько человек записалось
   - **🎲 Запустить жеребьёвку** — рандомно раздать кухни и задания
   - **🔓 Раскрыть задания** — после вечеринки показать всем кто что делал
   - **🔄 Сбросить** — начать заново

### Для участников:
1. Каждая пара пишет боту `/start`
2. Бот попросит ввести имена: `Макс и Катя` или `Артём`
3. После жеребьёвки каждый получит свою кухню и тайное задание
4. Кнопки **«Моя кухня»** и **«Моё задание»** чтобы не забыть

## Вариант 2: Запуск в облаке (Railway) — бот работает 24/7

Бот будет работать даже если выключить компьютер. Бесплатно.

### 1. Зарегайся на GitHub
Если нет аккаунта — зайди на https://github.com и создай.

### 2. Создай репозиторий на GitHub
1. Открой терминал и выполни:
```bash
cd /Users/maximkirnichniy/party-bot
git init
git add .
git commit -m "party bot"
```
2. Зайди на https://github.com/new
3. Назови репозиторий `party-bot`, нажми **Create repository**
4. Выполни команды которые покажет GitHub (будут типа):
```bash
git remote add origin https://github.com/ТВОЙ_ЮЗЕРНЕЙМ/party-bot.git
git branch -M main
git push -u origin main
```

### 3. Задеплой на Railway
1. Зайди на https://railway.app и войди через GitHub
2. Нажми **New Project** → **Deploy from GitHub repo**
3. Выбери свой `party-bot` репозиторий
4. После деплоя зайди в **Variables** (вкладка слева) и добавь:
   - `BOT_TOKEN` = твой токен от BotFather
5. Railway автоматически перезапустит бот — готово!

> Позже, когда узнаешь свой Telegram ID (бот покажет при первом /start),
> можешь добавить переменную `ADMIN_ID` = твой ID, чтобы при рестарте бот помнил кто админ.

## Если что-то не работает

- **Python не найден**: установи Python с https://python.org
- **Бот не отвечает**: проверь что токен правильный
- **Хочешь переназначить админа**: добавь переменную `ADMIN_ID` в Railway (или в `bot.py` при локальном запуске)
